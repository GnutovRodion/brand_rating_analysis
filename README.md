# Анализ успеваемости студентов

### Описание

Скрипт для формирования отчета рейтинга брендов.
Отчет строится по данным из CSV-файлов.
Архитектура позволяет легко добавлять новые отчёты:
достаточно добавить их в словарь REPORTS и написать 
класс для реализации функциональности, унаследовавшись от
BaseGeneratorReport.

### Запуск

Находясь в корневой директории проекта, выполните команду:

`python -m app.main --files {CSV-файлы для обработки} --report {название отчета}`

Пример:

`python -m app.main --files data/products1.csv data/products2.csv --report average-rating`

Скриншоты запуска скрипта в директории screenshots.

### Пример отчета

|    | brand   |   rating |
|----|---------|----------|
|  1 | apple   |     4.55 |
|  2 | samsung |     4.53 |
|  3 | xiaomi  |     4.37 |

### Запуск тестов

Находясь в корневой директории, выполните команду:

`pytest`

### Зависимости

- Python 3.12+
- Poetry
- tabulate
- pytest
- flake8

Для создания виртуального окружения и установки зависимостей выполните из корневой директории команду:

`poetry install`
